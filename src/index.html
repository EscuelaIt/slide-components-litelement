Prueba con componentes de efecto slide-down

Estos componentes están trabajados en el Curso de LitElement, en la clase dedicada a efectos de animación.

Tenemos tres componentes:

  • Menú responsive (tienes que tener una anchura de página pequeña para que aparezca)
  • Menú overlay. Que permite configurar un contenido que se muestra en una capa superpuesta
  • Componente "Ver más", que muestra un contenido parcialmente y tenemos un botón para "ver más" que permite mostrar más cosas.
